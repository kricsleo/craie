## Features

- 🚀 Tiny < 1kB
- 💪🏻 Full typescript support
- 📦 No dependencies
- 🪢 Elegant chain calls

## Usage

```bash
npm install craie
```

```ts
import craie from 'craie'

craie.log(craie.blue.bgRed.round('Message'))
```
